What is an HTML5 game engine?

  • The idea

  • Code once, multi-platform/multi-device support
  • Still gain access to certain hardware specific features:
    • Currently: gps & accelerometer Bleeding Edge: camera & microphone

  • How?

  • Leverage Canvas
  • Websockets
  • Localstorage/Offline

IMPACT

Features

  • Built in performance and debug tool
  • Good documentation (there is even an O'Reilly book!!1!) and forum
  • Homebrew level editor
  • Supports plugins, plugin resource
  • Box2D for physics (through plugin), fun demo
  • Socket.IO support (through plugins, impsock, impactConnect)
  • Commercial $99 :-(

Synopsis

It is commercial and closed with no way to try it free. The homebrew level editor doesn't support 2.5D 'isometric' games but can be faked. The maturity of the project gives to the good documentation and community, the developer previous two releases were May 2012 & December 2011, if there is a new major release you would need to purchase a new license according to the disclaimer. Overall, very solid engine.
